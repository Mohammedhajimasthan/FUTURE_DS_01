# Business Sales Performance Analysis

## 📊 Project Overview
This project analyzes business sales data to identify revenue trends, top-performing products, and regional performance.

## 🎯 Objective
- Analyze sales trends over time
- Identify top-selling products
- Evaluate category and regional performance
- Provide business insights and recommendations

## 🛠️ Tools Used
- Power BI
- Excel

## 📈 Key Insights
- Technology category generates the highest sales and profit.
- West region contributes the highest revenue.
- Sales show a declining trend over time.
- Furniture category has lower profit margins.

## 💡 Recommendations
- Focus on high-performing categories like Technology.
- Improve profitability in Furniture category.
- Expand strategies in underperforming regions.
- Address declining sales with targeted strategies.
